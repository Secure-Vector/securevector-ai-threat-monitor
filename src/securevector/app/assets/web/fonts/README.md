# Bundled fonts (v5 type signature)

Self-hosted so the app's CSP (`font-src 'self'`) is satisfied and the type
identity works fully offline — no external font CDN.

- **Space Grotesk** (500/600/700) — display / headings. © Florian Karsten.
- **Space Mono** (400/700) — metadata, numbers, telemetry readouts. © Colophon Foundry.
